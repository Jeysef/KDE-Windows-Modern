## Merge Strategy Recommendations

1. **Eleven + Cobalt as base:** Since Eleven is derived from Cobalt, treat them as the primary pair. Use Eleven's materialized copy (which includes Cobalt icons). This gives the strongest foundation.

2. **Win11 for app gap-filling:** When Eleven lacks a specific app icon, pull from Win11 (best app icons in the ecosystem). Win11's rounded-square status icons are also worth using selectively.

3. **Windows-Eleven for places/mimetypes overflow:** When both Eleven and Win11 lack a specific places or mimetype icon, Windows-Eleven's large set covers many gaps.

4. **Avoid Fluentwin for actions:** Its colorful actions are unusable in a toolbar context. Use only for devices or mimetypes if needed.

5. **Avoid Fluent as primary source:** Symbolic-only app icons and monochrome places make it unsuitable for most categories. Reserve for specific status icons that may be unique to Fluent.

6. **Skip Windows-Beuty and We10X:** These are pure clones. Use Windows-Eleven and Win11 respectively as their canonical equivalents.
